## Descripción del proyecto
Este proyecto corresponde a la Semana 3 de la asignatura Programación Orientada a Objetos.  
El objetivo es comparar dos formas de resolver un mismo problema en Python: la Programación Tradicional y la Programación Orientada a Objetos.

El sistema permite registrar y mostrar información básica de una mascota, considerando los siguientes datos:

- Nombre
- Especie
- Edad
